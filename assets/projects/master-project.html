Implementation of cyclic stress loading for small-scale material characterization

  • Tech Stack: motor control, LPBF specimen, fatigue testing, python

The project was conducted to characterize the behavior of the pantograph structure(Laser Powder Bed Fusion, LPBF specimen), which can support large deformation before failure. The characterization included torsion and cycling tests on two types of specimens with different geometry. Python was used to implement the cycling tests by controlling the motor in the testing device, and analyzing and visualizing the datasets.
